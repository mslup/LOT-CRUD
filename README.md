### System zarządzania lotami i pasażerami

## Wymagania
- Java 21
- Docker
- Gradle

## Uruchomienie
1. Sklonuj repo.
2. Użyj komendy:
```bash
docker compose up 
```
3. Aplikacja jest uruchumiona na porcie `http://localhost:8080`.

## Dokumentacja
Dokumentacja endpointów SwaggerUI znajduje się pod adresem `http://localhost:8080/swagger-ui/index.html`.
